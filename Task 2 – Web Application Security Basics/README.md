## Overview

This task demonstrates basic web application security testing using DVWA (Damn Vulnerable Web Application). The practical focuses on identifying and exploiting Cross-Site Scripting (XSS) vulnerabilities in a controlled environment.

The assessment includes:

* Reflected XSS
* Stored XSS
* DOM-Based XSS
* Vulnerability testing
* Security analysis
* Mitigation techniques

---

## Tools Used

* DVWA
* XAMPP
* Web Browser

---

## Vulnerabilities Tested

### Reflected XSS

Demonstrated client-side script injection through reflected user input.

### Stored XSS

Demonstrated persistent script injection stored on the server.

### DOM-Based XSS

Demonstrated browser-side script execution through DOM manipulation.
